    Compilation Commands:
    $ ./script.sh
    OR
    $ gcc -Wall -pedantic-errors lab03.c -o lab03.out
    Run Command:
    $ ./lab03.out <int1> <operation> <int2>
    where,
    <int1> and <int2> are binary numbers with up to 64 bits.
    <operation> is either -and representing bitwise-AND (&),
    -or representing bitwise-OR (|),
    or -xor representing bitwise-XOR (^).
